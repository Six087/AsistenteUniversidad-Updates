# Asistente Universidad - Actualizaciones

Repositorio público usado únicamente para distribuir actualizaciones de **Asistente Universidad**.

Este repositorio no contiene el código fuente principal del proyecto. Su función es servir como canal de actualización para que la aplicación pueda detectar nuevas versiones y descargar el ejecutable más reciente automáticamente.

## ¿Para qué sirve?

La aplicación consulta los **Releases** de este repositorio para saber si hay una versión nueva disponible.

Cuando existe una actualización, el programa puede descargar el archivo:

```text
AsistenteUniversidad.exe
