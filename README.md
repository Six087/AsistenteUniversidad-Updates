# ◈ Asistente Universidad - Actualizaciones

Repositorio público usado únicamente para distribuir actualizaciones de **Asistente Universidad**.

**Asistente Universidad** es una aplicación de escritorio creada como apoyo para estudiantes del **IES 9-012 de Informática**, de **San Rafael, Mendoza, Argentina**.

El programa ayuda a organizar materias, entregas, horarios, notas, archivos, recordatorios y datos académicos, con sincronización en Firebase para poder usarlo desde distintas PC.

---

## ◈ ¿Para qué sirve este repositorio?

Este repositorio funciona como canal de actualizaciones.

La aplicación consulta los **Releases** de este repositorio para detectar si existe una nueva versión disponible.  
Cuando hay una actualización, el programa puede descargar el ejecutable más reciente:

```text
AsistenteUniversidad.exe

```

---

## ◈ Seguridad y acceso

Aunque este repositorio sea público y el ejecutable pueda descargarse, el acceso al programa está protegido por un sistema de inicio de sesión.

Esto significa que descargar el archivo no permite usar la aplicación libremente: solo las personas autorizadas con una cuenta válida pueden acceder al sistema y a sus datos.

No se publican en este repositorio datos personales, sesiones, configuraciones privadas ni información sensible de la Base de Datos.

---

## ◈ Última versión

Las versiones oficiales se publican en:

[Releases](https://github.com/Six087/AsistenteUniversidad-Updates/releases)

Cada release puede incluir:

- Número de versión.
- Cambios principales.
- Mejoras o correcciones.
- Ejecutable actualizado.
- Archivo listo para descargar.

---

## ◈ Proyecto principal

El proyecto nació como una herramienta personal al comenzar la carrera de **Desarrollo de Software**, para resolver problemas de organización con materias, entregas y materiales de la plataforma académica.

Con el tiempo fue creciendo hasta convertirse en un asistente universitario más completo, con ayuda de herramientas de inteligencia artificial como **Codex** y **Claude Code**.

---

## ◈ Archivo esperado para actualizar

Para que el actualizador funcione correctamente, cada release debe incluir un archivo `.exe` con un nombre parecido a:

```text
AsistenteUniversidad.exe
```

La aplicación busca el ejecutable dentro del último release publicado.
